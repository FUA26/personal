## Description

Vascomm carrer merupakan portal karir dari PT. Vascomm Solusi Teknologi. Website ini terbagi menjadi 2 bagian dimana landing page untuk public dan backoffice yang digunakan oleh HR untuk manage kanidat pegawai. Pada project ini saya bertanggung jawab sebagai Full Stack Developer dan merupakan developer tunggal. Waktu pengerjaan dari website ini kurang lebih berjalan selama 2 bulan dan di rilis pada februari 2019. Sampai saat ini website masih berjalan dan dapat di akses pada link berikut [ini](https://vascomm.co.id/).
